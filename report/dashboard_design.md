## 2. 各图表详细设计

### 2.1 KPI 卡片 (关键指标)

| 指标 | 当前值 | 环比变化 | 可视化方式 |
|------|--------|---------|-----------|
| DAU（日活） | 2,983 | +5.2% | 数字卡片 + 趋势迷你图 |
| PV（浏览量） | 42,509/天 | +3.1% | 数字卡片 |
| 转化率 CVR | 2.03% | +0.1pp | 数字卡片 + 进度环 |
| 人均PV | 102.7 | -1.2% | 数字卡片 |
| 加购率 | 5.53% | +0.3pp | 数字卡片 |
| 购买用户数 | 7,102 | +8.5% | 数字卡片 |

**适用工具：**
- **Power BI**: 使用 Card visual 或 Multi-row card
- **Tableau**: 使用 BAN (Big Number) 或 Text table
- **Excel**: 使用单独的单元格区域或 Sparkline 迷你图

### 2.2 DAU 趋势图

| 属性 | 说明 |
|------|------|
| 图表类型 | 折线图 (Line Chart) |
| X轴 | 日期（连续） |
| Y轴 | DAU 数值 |
| 辅助线 | 7日移动平均线（MA7） |
| 交互 | 日期范围筛选器、悬停显示数值 |
| 颜色 | 主线条 #5B9BD5，MA7线 #FFC000 |

**适用工具分析：**
| 工具 | 评价 |
|------|------|
| **Power BI** | 原生折线图，支持趋势线、预测线，适合周期性监控 |
| **Tableau** | 日期层级自动下钻（年/季/月/周），交互式筛选体验更好 |
| **Excel** | 图表操作简单，适合快速生成静态报告，不支持交互 |

### 2.3 漏斗图 (Funnel)

| 属性 | 说明 |
|------|------|
| 图表类型 | 漏斗图 / 瀑布图 |
| 阶段 | PV → FAV → CART → BUY |
| 核心数据 | 各阶段数量、阶段转化率、累计流失率 |
| 颜色 | 渐变色（蓝→绿） |

**适用工具分析：**
| 工具 | 评价 |
|------|------|
| **Power BI** | 无原生漏斗图，需使用自定义视觉（如 Chiclet Slicer 或第三方漏斗插件） |
| **Tableau** | 内置 Funnel 图类型，可直接使用；或通过堆积条形图模拟 |
| **Excel** | 可使用堆积条形图手动制作漏斗，需较多手工作业 |

### 2.4 留存热力图 (Retention Heatmap)

| 属性 | 说明 |
|------|------|
| 图表类型 | 热力图 (Heatmap) + 表格 |
| 行 | 获客周 (Acquisition Week) |
| 列 | 周数偏移 (Week 0, 1, 2, ...) |
| 数值 | 留存率百分比 (%) |
| 颜色 | 渐变色（白→黄→红→深红） |

**适用工具分析：**
| 工具 | 评价 |
|------|------|
| **Power BI** | 使用 Matrix 视觉 + 条件格式颜色刻度，效果最佳 |
| **Tableau** | 内置 Heatmap 图类型，颜色梯度控制和交互最灵活 |
| **Excel** | 条件格式 + 数字矩阵可实现，适合静态报告但交互性差 |

### 2.5 用户画像 (User Segmentation)

| 属性 | 说明 |
|------|------|
| 图表类型 | 饼图 / 环形图 + 堆积柱状图 |
| 维度 | 用户分层（高价值/潜力/沉默/流失） |
| 指标 | 用户数占比、各层级的转化率 |

**适用工具分析：**
| 工具 | 评价 |
|------|------|
| **Power BI** | 饼图 + Slicer 联动，可做钻取到细分用户 |
| **Tableau** | 饼图/圆环图配合 Sets 做用户分组分析 |
| **Excel** | 饼图制作简单，适合静态呈现 |

### 2.6 热门商品 & 品类 (Top 10)

| 属性 | 说明 |
|------|------|
| 图表类型 | 横向柱状图 (Bar Chart) |
| 排序 | 按总行为量降序排列 |
| 颜色 | 按品类分别配色 |

**适用工具分析：**
| 工具 | 评价 |
|------|------|
| **Power BI** | Top N 筛选器内置支持，动态排名切换 |
| **Tableau** | 索引计算字段实现 Top N，排序交互性好 |
| **Excel** | 使用 LARGE() 函数或数据筛选，手动更新 |

### 2.7 时间热力图 (Hour x Weekday)

| 属性 | 说明 |
|------|------|
| 图表类型 | 热力图 (Heatmap) |
| 行 | 星期一 ~ 星期日 |
| 列 | 0~23 小时 |
| 数值 | 行为量 / 购买量 |

**适用工具分析：**
| 工具 | 评价 |
|------|------|
| **Power BI** | Matrix + 颜色条件格式，操作最简便 |
| **Tableau** | Heatmap 图类型，颜色梯度自然，交互最流畅 |
| **Excel** | 条件格式 + 颜色矩阵，适合静态报告 |

## 3. 工具选择总评

| 需求 | 推荐工具 | 理由 |
|------|---------|------|
| **日常监控** | Power BI | 自动刷新、数据源集成好、移动端友好 |
| **深度探索** | Tableau | 交互层级丰富、视觉美观、钻取分析灵活 |
| **周报/月报** | Excel | 上手简单、格式控制强、团队协作方便 |
| **PPT汇报** | Excel → PPT | 图表复制粘贴便利、格式一致性强 |

## 4. 数据刷新策略

| 层级 | 刷新频率 | 数据源 | 说明 |
|------|---------|--------|------|
| KPI 卡片 | 每天凌晨 | 数仓汇总表 | T+1 数据 |
| DAU 趋势 | 每天凌晨 | 数仓日活表 | 保留90天 |
| 漏斗/留存 | 每周一 | 数仓行为表 | 周级汇总 |
| 商品排名 | 每天凌晨 | 实时数仓 | T+1 数据 |
| 用户画像 | 每月初 | 用户标签表 | 月度更新 |
